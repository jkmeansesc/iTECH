To run the code:
1. python manage.py makemigrations
2. python manage.py migrate
3. python populate_authentication.py
4. python manage.py runserver

There are four preset accounts:
1. staff accounts:
   user1(username) - user1(password)
   user2 - user2
   user3 - user3
2. normal account:
   user4 - user4
