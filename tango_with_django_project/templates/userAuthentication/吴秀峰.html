  • 标题： 请教下django查询问题，开始时间2023-7-15 00:00:00 结束时间2023-7-20 23:59:59，查询统计出每天的数据总和，并输出这几天每天的总和 2024-01-06

    内容：from django.db.models import Sum from datetime import datetime, timedelta start_date = datetime(2023, 7, 15) end_date = datetime(2023, 7, 20) + timedelta(days=1) date_range = [] current_date = start_date while current_date &lt; end_date: date_range.append(current_date) current_date += timedelta(days=1) querysets = [] for date in date_range: querysets.append(MyModel.objects.filter(date_field__range=[date, date+timedelta(days=1)]).values(&#x27;date_field&#x27;).annotate(total=Sum(&#x27;my_field&#x27;))) for queryset in querysets: for result in queryset: print(result[&#x27;date_field&#x27;], result[&#x27;total&#x27;])
